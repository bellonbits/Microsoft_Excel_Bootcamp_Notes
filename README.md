## About This Course

This 4-day Excel Bootcamp is designed to take participants from the basics of Excel all the way through to professional-level data analysis tools. Whether you are new to Excel or looking to sharpen your existing skills, this course provides hands-on training with real-world examples across every major area of the software.

By the end of the bootcamp, participants will be confident using Excel to manage, analyse, visualize, and present data effectively in any professional setting.

---

## Who Is This Course For?

This course is suitable for anyone who works with data or wants to improve their productivity using Excel. It is ideal for students, analysts, business professionals, administrators, and anyone looking to build a strong foundation in spreadsheet skills.

No prior Excel experience is required, though a basic familiarity with computers is assumed.

---

## What You Will Learn

Over four days, participants will cover the full breadth of Excel's core functionality, including:

- Navigating Excel and understanding its key terminology
- Managing and manipulating data in cells and ranges
- Using Excel's most important built-in functions for calculations and text
- Writing logical formulas to automate decisions
- Applying conditional formatting and data validation to improve data quality
- Organising data using Excel Tables
- Creating and customising charts and graphics
- Building PivotTables and PivotCharts for data summarisation
- Performing What-If Analysis to model different scenarios
- Cleaning and standardising data for accurate reporting

---

## Course Structure

### Day 1 — Excel Foundations & Data Management
Participants are introduced to Excel's core concepts and interface. This day covers the essential terminology every Excel user needs to know, followed by practical skills in sorting, filtering, and working with formulas to manage data efficiently.

### Day 2 — Excel Functions
This day is dedicated to Excel's most powerful built-in functions. Participants will learn calculation functions, text manipulation functions, and lookup functions — the tools used most frequently by data professionals in everyday work.

### Day 3 — Logic, Formatting & Tables
Participants learn how to write logical formulas that make Excel respond intelligently to data. The day also covers how to use conditional formatting to highlight important information, data validation to control data entry, and Excel Tables to manage structured datasets.

### Day 4 — Charts, PivotTables & Data Cleaning
The final day focuses on communicating data visually through charts and PivotTables, exploring scenarios with What-If Analysis tools, and ensuring data quality through practical data cleaning techniques.
